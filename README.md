# ⚡ Usage

|Keyboard        |Functionality|
|----------------|-------------|
|<kbd>  ↑  </kbd>|Move up      |
|<kbd>  ↓  </kbd>|Move down    |
|<kbd>  ←  </kbd>|Move left    |
|<kbd>  →  </kbd>|Move right   |
|<kbd>space</kbd>|sweeper      |
|<kbd>  Q  </kbd>|Quit         |
